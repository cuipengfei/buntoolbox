# Buntoolbox

> **Bun** + **Ubuntu** + **Toolbox** = 全能开发环境 Docker 镜像

## 包含组件

- **运行时**: Bun, Node.js, Python
- **JDK**: 11, 17, 21
- **基础镜像**: Ubuntu
- **常用工具**: 内置开发必备工具集

## 使用方式

```bash
docker pull yourname/buntoolbox:latest
docker run -it yourname/buntoolbox
```

## 命名由来

| 组合 | 含义 |
|------|------|
| Bun | 现代 JS 运行时 |
| (U)buntu | 稳定的 Linux 基底 |
| Toolbox | 多语言工具箱 |
