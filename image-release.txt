# Image Variants
- latest: terminal/TUI development image
  - Build file: Dockerfile
  - Base image: ubuntu:26.04
  - Primary ports: 22/tcp SSH, 7681/tcp ttyd
  - Default workflow: root shell and terminal-first development tools
- i3: browser-delivered i3 desktop image
  - Build file: docker/webtop/Dockerfile with WEBTOP_BASE=lscr.io/linuxserver/webtop:ubuntu-i3
  - Base image: lscr.io/linuxserver/webtop:ubuntu-i3
  - Primary ports: 3200/tcp webtop GUI HTTP, 3201/tcp webtop GUI HTTPS, 7681/tcp ttyd
  - Default workflow: root-first interactive desktop; an abc account may exist for upstream compatibility but is not the normal workflow
- kde: browser-delivered KDE desktop image
  - Build file: docker/webtop/Dockerfile with WEBTOP_BASE=lscr.io/linuxserver/webtop:ubuntu-kde
  - Base image: lscr.io/linuxserver/webtop:ubuntu-kde
  - Primary ports: 3200/tcp webtop GUI HTTP, 3201/tcp webtop GUI HTTPS, 7681/tcp ttyd
  - Default workflow: root-first interactive desktop; an abc account may exist for upstream compatibility but is not the normal workflow
  - Desktop note: KDE Ubuntu Webtop is Wayland-only upstream and heavier than i3
